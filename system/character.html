# 創造角色

Night of the Dead 之中的每個人都知道這個小鎮的夜晚是相當地危險，甚至會無法平安脫身。 因此角色都必須要有一個非常強烈的願望或動機。

# 建立角色形象的問題

  • （外觀）你給別人的第一印象？
  • （目標）你為何想要進入這個小鎮？
  • （代價）你願意以什麼樣的代價來取得目標？
  • （信心）你有什麼自信可以在這個小鎮存活？
  • （期望）你預期在這個小鎮找到什麼？
  • （害怕）你最擔心的事情是什麼？
  • （秘密）你心裡藏了什麼不能讓人知道的事情？
